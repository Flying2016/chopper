# Chopper
> 一个shell集合库


### 简介
- 只是一个shell脚本集合
- 方便自己使用，处理日常工作

### 如何使用？
> 执行命令之后，开始下载这个脚本，这个脚本会反向回来根据用户选择pull scripts中的文件并执行
```bash
curl -o- https://raw.githubusercontent.com/owen-carter/chopper/master/bin.sh | bash
```
